# 🛡️ 병역준비 건강처방 AI

> **2026 질병관리청 · 병무청 · 방위사업청 합동 공공데이터·AI 활용 경진대회 출품작 (MVP)**

세 주관기관을 하나의 스토리로 잇는 청년 건강관리 서비스입니다.

| 기관 | 역할 | 앱에서의 반영 |
|------|------|--------------|
| **질병관리청** | 국민건강영양조사(KNHANES) | 국가 평균 대비 내 건강 벤치마크 |
| **병무청** | 병역판정·입영 관리 | 입영까지 기간 반영 맞춤 건강 처방 |
| **방위사업청** | 국방 R&D·스마트국방 | 국방 웨어러블 헬스 데이터 확장 시나리오 |

## 주요 기능
1. **내 건강 진단** — 같은 연령대 국가 평균/분포 대비 BMI·혈압·체중 벤치마크
2. **맞춤 처방** — 입영까지 남은 기간과 체력 수준을 반영한 주간 운동 플랜 + 군 체력검정 대비 체크리스트
3. **국방 웨어러블** — 웨어러블 헬스 데이터가 입대 후 병사 건강관리로 이어지는 확장 시나리오 (방사청 연계)
4. **데이터·근거** — 출처, 방법론, 면책 고지

## 실행 방법
```bash
pip install -r requirements.txt
streamlit run app.py
```
기본 접속: http://localhost:8501

## 프로젝트 구조
```
app.py             # Streamlit UI (4개 탭)
health_logic.py    # BMI·신체등급(참고용)·주간 처방 로직
synthetic_data.py  # KNHANES 근사 합성데이터 + 웨어러블 시계열 생성
requirements.txt   # 의존성
.streamlit/config.toml
```

## 데이터 고지
- 본 MVP는 질병관리청 공개 요약통계 범위를 참고한 **합성(synthetic) 샘플 데이터**를 사용합니다. **KNHANES 원자료가 아닙니다.**
- 실서비스화 시 KNHANES 이용승인 원자료 또는 공공데이터포털 API로 대체 가능합니다.

## ⚠️ 면책
본 앱의 신체등급은 병역판정 신체검사의 **체격(BMI) 기준만 단순화한 참고값**입니다. 실제 병역판정은 병무청이 수백 개 항목을 종합해 결정하며 본 결과와 무관합니다. 의료 진단을 대체하지 않습니다.
